About

We are FTC Team 4654 "The Jellyfish" from Blind Brook High School, New York. We participate in the FIRST Tech Challenge, an annual robotics competition in which teams compete for awards and placement to Worlds.
Meet the Team About Us

Engineering

Every year, we design and build a new robot for the FIRST Tech Challenge competition. After brainstorming, we use computer aided design (CAD) to model our robot digitally. Then, we physically build the robot before testing to see its flaws. We then adjust and repeat. This process is documented in our Engineering Portfolio, which we release publicly.

CAD Notebook

Programming

For the yearly FIRST Tech Challenge, we write code for the autonomous and driver-controlled periods of the game, which we release to the public. We also provide documentation and explanations for algorithms we use, as well as a robot simulator.
